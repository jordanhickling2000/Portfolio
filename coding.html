    This is the subquery I have made and used:
    "SELECT emp_id, emp_name, job_name, commission FROM employees WHERE emp_id IN (SELECT emp_id FROM employees WHERE commission IS NOT NULL);"

    With this subquery the objective is to make the table only get the commissions that are not empty in the table. This means that the only 4 that should display are "ADELYN", "WADE", "MADDEN", "TUCKER". You could go further and use "AND" in the where clause and say "commission > 0". This would then kick out "TUCKER" and display just 3 names.

    The Query SELECTS and shows only the employee ID (emp_id), the Employees Name (emp_name), the job name(job_name) and the commission from the employees table (what you see above) It then searches for the employee ID IN the subquery that is written afterwards that selects the emp_id which is what it will get following the where clause. The where clause searches for the commissions that have cells filled. "IS NOT NULL" means that it will return with cells that are not empty.
